# Adoption Signal Review Template

Use this template in recurring adoption reviews, rollout check-ins, retrospectives or scaling discussions.

The goal is not only to review numbers.

The goal is to understand what the signals reveal about the work system and decide what should happen next.

## 2. Review question

Start with the central question:

```text
What did we learn about the work system?
```

Optional supporting questions:

```text
Where is adoption becoming visible?
Where is friction still present?
Where are people using the new capability appropriately?
Where are signals ambiguous?
Where is value emerging?
Where is risk becoming visible?
Which decision is needed next?
```
